Text Colours

Text should appear as Jet Grey in most cases.

Pure White may be used for contrast with dark backgrounds.

Typography

All fonts used by Hundred Hats are free and open source web fonts.

Headings

Headings should be written in 700 weight Roboto.

Significant headings should be 2.5x the font size of the body text.

Subheadings should be 1.5x the font size of the body text.

Body Text

Body text should be written in 300 weight 18px Roboto.

Bolded text should appear as 400 weight 18px Roboto.

Code Blocks

// Code blocks should appear in 300 weight 18px Roboto Mono.
